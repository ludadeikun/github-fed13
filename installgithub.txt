получение актуальной версии проекта

git fetch   - получить с удаленного репозитория последние версии файлов

git pull    - выгрузить полученные файлы в папку с копией проекта

создать на github запрос на добавление изменений (pull requests)


-----------------------------------------------------------------------------------------
Базовые операции c github

git status  (текущее состояние файлов с изменениями)

git add .     (добавляет изменненые файлы в локальный репозиторий git)

git commit  -m"add file"   (фиксирует слепок/снимок файлов на текущий момент)

git push   (синхронизирует/отправляет файлы на удаленный репозиторий)


-----------------------------------------------------------------------------------------
Командная работа

git branch -v (показывает список веток)

git branch 

git checkout имяветки  (переключение на конкретную ветку)

git checkout -b branch3 (создание и переход в созданную ветку)

---------------------------------------------------------------------------------------

возврат локального репозитория до состояния опредленного коммита

git log
q - выход из просмотра log
git checkout индекс комита  название файла

---------------------------------------------------------------------------------------

возврат репозитория до указанного комита (для github)
git log - просматриваем все комиты и находим нужный
q - выход из просмотра log
$ git checkout -- <файл>
$ git push -f  синхронизация изменений в github несмотря на версию
